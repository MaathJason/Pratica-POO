# POO-Pratica

Projeto desenvolvido durante o curso **Praticando C# - Orientação a Objetos** da Alura.

## Descrição

Esta solução reúne diversos projetos independentes que simulam sistemas de diferentes domínios, utilizando conceitos de Programação Orientada a Objetos (POO) em C#. O objetivo é praticar encapsulamento, composição, propriedades, listas e métodos, aplicando boas práticas de orientação a objetos.

## Funcionalidades

- **Biblioteca**
  - Representação de livros com título, autor e ano.
  - Exibição de informações detalhadas dos livros.

- **Cálculo de Área**
  - Cálculo de área e perímetro de retângulos a partir de largura e altura.

- **Controle de Estoque**
  - Controle simples de estoque de produtos.
  - Operações de compra, venda e atualização de preço.

- **Emissão de Passagens**
  - Modelagem de passagens com origem, destino, data e preço.
  - Exibição de resumo das passagens.

- **Gestão de Funcionários**
  - Representação de funcionários com nome, cargo e salário.
  - Operações de aumento e exibição de dados.

- **Loja de Software**
  - Modelagem de produtos digitais com informações técnicas associadas.
  - Exibição de detalhes dos produtos.

- **Recomendação de Filmes**
  - Representação de filmes, registro de avaliações e exibição de recomendações.

- **Sistema Bancário**
  - Operações bancárias básicas: depósito, saque e transferência entre contas.

- **Sistema de Clínica**
  - Modelagem de consultas médicas com paciente, data, horário e descrição.
  - Exibição de agendamentos.

- **Sistema de Pedidos**
  - Criação de pedidos, adição de itens/quantidades e cálculo do total.

## Tecnologias Utilizadas

- C# 12.0
- .NET 8

## Estrutura Principal

- Cada projeto contém uma ou mais classes que modelam o domínio do exercício (por exemplo, 'Livro', 'Produto', 'ContaBancaria').
- Os arquivos 'Program.cs' servem como demonstração, mostrando a criação de objetos e chamadas de métodos.
- O repositório está organizado como uma solução com projetos independentes.

## Como Executar

1. Clone o repositório.
2. Abra a solução no Visual Studio 2022.
3. Compile e execute o projeto desejado.

## Aprendizados

- Criação e uso de classes e objetos em C#.
- Encapsulamento de dados com propriedades.
- Composição de objetos e relacionamento entre classes.
- Manipulação de listas e métodos.
- Aplicação de boas práticas de POO.
